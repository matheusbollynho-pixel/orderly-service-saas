## 🔍 Se Algo Não Funcionar

### Toast não aparece
**Solução:** Verifique no Debug → Card "Tabela de Palavras-chave"
- Se mostrar 0 → Crie uma palavra-chave
- Se mostrar > 0 → Verifique cliente autorização

### Lembrete não cancela
**Solução:** Abra DevTools (F12) → Console
- Procure por: "✅ N lembrete(s) cancelado(s)"
- Se não aparecer → Lembrete já foi enviado (então cria novo)

### Dashboard não atualiza
**Solução:** Pressione F5 (atualizar página)
